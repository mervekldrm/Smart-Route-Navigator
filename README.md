# Smart Route Navigator

## Description
This web application finds and displays the shortest path between two nodes selected on a map using Dijkstraâ€™s Algorithm and Leaflet.js.

## Setup
1. Run a local server:
   - Python: `python -m http.server 8000`
   - Or use Visual Studio Code "Live Server" extension.
2. Open your browser at `http://localhost:8000`.
3. Select two nodes on the map to see the shortest path!
